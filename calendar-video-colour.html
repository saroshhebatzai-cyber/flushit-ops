The smallest step from black — clearly blue up close, still authoritative from across the room. Safest if you like the current weight and only want it to stop being literally black.
My pick: A, steel blue #3D5A80.
It's the only option that fixes the actual problem rather than just softening it. Black currently makes every video deadline look more urgent than an overdue project; steel blue puts videos in their own calm lane and lets red/amber own urgency. It's also already a token in the old OS palette, and it's the one hue no client colour is using — so the dot inside the chip always stays legible.

If you want to keep the current heft, C is the conservative version of the same idea. I'd skip B — a second green next to sage muddies the health colours.
